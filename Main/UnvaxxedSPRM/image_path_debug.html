Debugging Tips

  • Ensure the file structure in your GitHub repository matches exactly the path specified in the src attribute.
  • GitHub Pages hosting is case-sensitive. Check for mismatched cases in folder and file names.
  • If using GitHub Pages, ensure that the images are pushed to the repository and that the branch hosting the site (e.g., gh-pages) includes the image files.
  • Use your browser's Developer Tools (F12) to inspect the network tab and verify if the images are being requested correctly or returning 404 errors.
  • If hosting through GitHub Pages, remember paths should be relative to the repository root unless a base URL is specified.
